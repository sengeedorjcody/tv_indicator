🎯 Inducement-ийн товч утга

Inducement гэдэг нь зах зээлийн том тоглогчид (smart money) бага арилжаачдыг алдагдалд оруулахын тулд богино хугацаанд хууран мэхэлж, дараа нь үнэ өөрсдийн жинхэнэ чиглэлдээ үргэлжлүүлэн хөдөлгөдөг үзэгдэл юм.

Өөрөөр хэлбэл: үнэ түр зуур өмнөх бага high/low-г эвдэж, арилжаачдыг өөр чиглэлд оруулж, дараа нь эргэж өмнөх чиглэлдээ хөдөлнө. Энэ нь жижиг арилжаачдын ликвидность (stop loss) "арчих" арга юм.

📈 График дээрх жишээ

🟢 Long (BUY) оролтын жишээ

Доорх график дээр үнэ түр зуур бага low-г доошоо эвдэж (inducement sweep), дараа нь дээшээ эргэж BUY signal өгсөн байна.

  1. ① Өмнөх swing low болон swing high үүссэн.
  2. ② Үнэ swing high-г эвдэж BOS өгч, bullish structure батлагдсан.
  3. ③ Үнэ буцаж бууж, бага swing low-г доошоо эвдэж inducement sweep хийсэн.
  4. ④ Дээшээ хаагдсан бөгөөд bullish FVG + OB баталгаатай BUY signal өгсөн.
  5. ⑤ SL inducement low-ын доор, TP 1:2 эсвэл 1:3 харьцаагаар тохируулна.

🔴 Short (SELL) оролтын жишээ

Доорх график дээр үнэ түр зуур бага high-г дээшээ эвдэж (inducement sweep), дараа нь доошоо эргэж SELL signal өгсөн байна.

  1. ① Өмнөх swing high болон swing low үүссэн.
  2. ② Үнэ swing low-г эвдэж BOS өгч, bearish structure батлагдсан.
  3. ③ Үнэ буцаж өсөж, бага swing high-г дээшээ эвдэж inducement sweep хийсэн.
  4. ④ Доошоо хаагдсан бөгөөд bearish FVG + OB баталгаатай SELL signal өгсөн.
  5. ⑤ SL inducement high-ын дээр, TP 1:2 эсвэл 1:3 харьцаагаар тохируулна.

✅ Signal өгөхийн өмнө биелэх нөхцөлүүд

🟢 Long (BUY) Signal-ийн нөхцөлүүд

  1. Market Structure bullish байх — үнэ өмнөх swing high-г эвдэж дээшээ гарсан (BOS эсвэл CHoCH үүссэн).
  2. Inducement sweep болох — үнэ бага swing low-г доошоо эвдэж, арилжаачдыг sell-д оруулж, дараа нь дээшээ хаагдана.
  3. FVG (Fair Value Gap) баталгаа — bullish FVG (gap up) үүссэн байх.
  4. Order Block (OB) баталгаа — ойролцоо bullish order block байх.
low < induceLow[1] // өмнөх бага low-г доошоо эвдсэн
close > induceLow[1] // дээшээ хаагдсан
bullish FVG үүссэн
bullish OB ойролцоо байна

🔴 Short (SELL) Signal-ийн нөхцөлүүд

  1. Market Structure bearish байх — үнэ өмнөх swing low-г эвдэж доошоо орсон (BOS эсвэл CHoCH үүссэн).
  2. Inducement sweep болох — үнэ бага swing high-г дээшээ эвдэж, арилжаачдыг buy-д оруулж, дараа нь доошоо хаагдана.
  3. FVG (Fair Value Gap) баталгаа — bearish FVG (gap down) үүссэн байх.
  4. Order Block (OB) баталгаа — ойролцоо bearish order block байх.
high > induceHigh[1] // өмнөх бага high-г дээшээ эвдсэн
close < induceHigh[1] // доошоо хаагдсан
bearish FVG үүссэн
bearish OB ойролцоо байна

⚙️ Indicator-ийн үндсэн тохиргоонууд

🔧 Swing / Structure Settings

  • Swing Lookback: Том swing high/low таних барын тоо. Жишээ: 5
  • Show Swing Highs/Lows: График дээр swing цэгүүдийг харуулах.
  • Show BOS / CHoCH: Бүтцийн эвдэлтүүдийг харуулах.

🔧 Inducement Settings

  • Detect Inducement: Inducement илрүүлэх ON/OFF.
  • Inducement Swing Lookback: Бага swing high/low таних барын тоо. Жишээ: 3

🔧 Entry Settings

  • Use FVG Filter: Fair Value Gap шүүлтүүр ашиглах.
  • Min FVG Size: Хамгийн бага FVG хэмжээ (pips).
  • Use Order Block Filter: Order Block шүүлтүүр ашиглах.
  • Risk:Reward: Ашиг алдагдлын харьцаа. Жишээ: 2.0
  • ATR Multiplier: Stop loss-ийг ATR-ээс хэдэн дахин томруулах. Жишээ: 1.5
